# .govulncheck-allow.txt — govulncheck vulnerability allowlist
#
# Format (one entry per line):
#   GO-YYYY-NNNN reachability=called|imported # owner: @handle — justification; tracked: ISSUE-123
#
# Fields:
#   GO-YYYY-NNNN          Go vulnerability database identifier (required)
#   reachability=called   The vulnerable function is in the call graph
#   reachability=imported The vulnerable package is imported but not called
#
# Rules:
#   - Every entry requires a justification comment and an owner.
#   - Stale entries (where the vulnerability no longer appears in findings)
#     cause the CI check to fail — remove them promptly.
#   - Entries are reviewed quarterly.
